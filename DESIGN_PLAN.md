# Run Witch Run - Website Design Plan

## Design Philosophy: Playful Enchantment with Modern Polish

This website celebrates a whimsical endless runner game with a design that balances **magical charm** with **modern web aesthetics**. The design uses vibrant purples, oranges, and mystical elements to reflect the game's witch-themed setting while maintaining clean, professional web standards.

### Core Design Principles:

1. **Magical Atmosphere**: Purple and orange color scheme reflecting the game's mystical night-time setting with glowing moon
2. **Playful Energy**: Rounded elements, smooth animations, and dynamic layouts that feel alive and engaging
3. **Clear Hierarchy**: Strong typography contrast between display and body text to guide user attention
4. **Responsive Delight**: Thoughtful mobile-first design with touch-friendly interactive elements
5. **Game-Centric**: Hero section prominently features the game with easy access to play

### Color Palette:

- **Primary Purple**: #6B46C1 (deep mystical purple)
- **Accent Orange**: #F97316 (warm glowing orange)
- **Dark Background**: #0F172A (deep night sky)
- **Light Text**: #F8FAFC (soft white)
- **Muted Purple**: #8B5CF6 (lighter purple for accents)

### Typography System:

- **Display Font**: Bold, playful serif or geometric sans-serif for headings
- **Body Font**: Clean, readable sans-serif (system fonts or Google Fonts)
- **Hierarchy**: H1 (48px), H2 (36px), H3 (24px), Body (16px)

### Layout Paradigm:

- **Hero Section**: Full-width with game embedded, creating immediate engagement
- **Feature Cards**: 3-column grid showcasing game features with icons
- **Asymmetric Sections**: Alternating left/right layouts for visual interest
- **Sticky Navigation**: Always accessible header for easy game access

### Signature Elements:

1. **Glowing Borders**: Subtle purple/orange glows on interactive elements
2. **Animated Icons**: Subtle floating/pulsing animations on feature icons
3. **Gradient Overlays**: Purple-to-orange gradients on buttons and highlights
4. **Witch Theme**: Moon, stars, and magical particle effects in backgrounds

### Interaction Philosophy:

- **Smooth Transitions**: All interactions use 300-400ms easing
- **Hover Effects**: Buttons scale slightly, cards lift with shadows
- **Click Feedback**: Immediate visual response to user actions
- **Loading States**: Animated spinners with game-themed styling

### Animation Guidelines:

- Entrance animations: Fade-in + slight slide-up (400ms)
- Hover animations: Scale transform (200ms ease-out)
- Button press: Brief scale-down (100ms) then scale-up
- Floating elements: Subtle Y-axis movement (3s infinite)

### Sections:

1. **Header/Navigation**: Logo, nav links, Play button
2. **Hero Section**: Game embedded, tagline, call-to-action
3. **Features Section**: Game mechanics showcase (3 cards)
4. **How to Play**: Simple instructions with screenshots
5. **Gallery**: Game screenshots carousel
6. **Footer**: Links, social, copyright

## Implementation Notes:

- Use React components for modular structure
- Tailwind CSS for styling with custom theme colors
- Framer Motion for animations
- Responsive breakpoints: mobile (320px), tablet (768px), desktop (1024px)
- Accessibility: WCAG 2.1 AA compliance, keyboard navigation support
